# 设置变灰 / 锁死诊断 / "Why is this setting grayed out?"

摄影师常遇到："快门不能调""ISO 变灰""眼控对焦开不了""触控追踪用不了"。**99% 的情况不是故障，而是上游某个模式/格式把这个参数接管或禁用了。** 诊断的关键是**先找出当前处于什么模式**，再对症解锁。

回答要**非常直接**：一句话点出最可能的锁定源 + 怎么解锁。格式：
> 当前处于 **XX 模式**，所以 YY 受限。**先退出 XX / 改成 ZZ，再调 YY。**

主要针对 Sony(FX30 / Alpha 线，社团在用)，逻辑对其他品牌通用，菜单名各家不同。

---

## 第 0 步：先问/先看这几项(快速定位锁定源)

让摄影师确认(或你据机型推断)：

1. **模式转盘**在哪？(Auto / P / A / S / M)
2. **S&Q**(慢动作&快动作)是否开启？
3. 当前**录制格式 / 帧率**？(尤其是否 100p/120p/240p 高帧率)
4. **曝光控制**是 Cine EI 还是 Flexible ISO / 自动？(FX 线特有)
5. **代理录制(Proxy)**是否开启？
6. **对焦模式**是 AF 还是 MF？**被摄体识别**开了吗？

下面把"症状 → 最可能原因 → 解锁"列出。

---

## 症状 → 原因 → 解锁

### 快门不能调 / 变灰
- **最可能：不在 M 或 S 档。** 在 P/Auto 下快门由相机定，A(光圈优先)下也不能手动设快门 → **转到 S 或 M 档**。
- **S&Q 模式**：慢动作下快门范围被帧率限制、且常锁定 → **退出 S&Q 再调**，或在 S&Q 设置里改"录制帧率"。
- **高帧率(100p/120p/240p)**：可用快门范围被帧率下限卡住(快门不能慢于 1/帧率) → 降帧率或在允许范围内设快门。
- **自动模式(Auto/iAuto)**：一切自动 → 退出到 P/A/S/M。

### ISO 变灰 / 不能设具体值
- **Cine EI 模式(FX30 等)**：ISO 固定在基准感光度，你调的是 **Exposure Index(EI)** 而不是 ISO → 想直接设 ISO 就切到 **Flexible ISO / 自定义模式**。
- **Auto ISO 开着**：ISO 交给相机，所以设不了固定值 → 关掉 Auto ISO 改手动 ISO。
- **某些自动曝光模式**：P/Auto 下 ISO 也可能自动。

### 人眼识别 / Eye AF 开不了
- **对焦模式是 MF**：手动对焦下没有 AF 识别 → 改 **AF-C/AF-S**。
- **被摄体识别没开 / 选错类别**：菜单里"AF 时人脸/眼睛优先"关了，或"识别对象"设成了动物/鸟 → 打开人脸/眼睛优先、识别对象选"人物"。
- **某些录制格式/高帧率**：HFR、部分高码率格式会禁用或限制识别 → 退出该格式。

### 触控追踪 / 触摸对焦用不了
- **触摸操作总开关关了**：菜单"触摸操作"或"触摸面板/触摸板"为关 → 打开。
- **"触摸功能"设成了对焦以外**(如仅触摸快门/触摸变焦关) → 设为"触摸追踪 / 触摸对焦"。
- **对焦模式 MF**：追踪需要 AF → 改 AF-C。
- **高帧率 / 特殊模式下**触控追踪可能被禁用。

### 录制格式 / 帧率 / 码率 选项变灰
- **代理录制(Proxy)冲突**：某些高帧率/高码率(如 4K120、All-Intra)不支持同时录代理 → 关掉 Proxy 才能选这些格式。
- **S&Q 开着**：会限制可选的主格式/帧率 → 退出 S&Q。
- **录制设置不兼容**：先定"文件格式"(XAVC HS/S/S-I)，再选"录制帧率/画质"，顺序错会显示变灰。

### Picture Profile / 创意外观变灰
- **Cine EI / Log 模式**：强制使用 S-Log3 等,创意外观(Creative Look)被禁用 → 退出到 Flexible ISO/自定义模式才能用直出风格。
- **HLG(HDR)模式**：会禁用部分 Picture Profile 与创意外观。

### 音频录制变灰 / 没有声音选项
- **高帧率/超慢动作(S&Q)**：高帧率慢动作通常**不录音** → 这是正常限制，要收音就别用 S&Q/HFR。

## 直接输出示例 / Direct answers

> **症状**：快门不能调、ISO 变灰、眼控开不了、触控追踪用不了(Sony FX30)
> **判断**：这几项同时受限，最典型的就是 **S&Q(慢动作)模式**——它会锁快门范围、限制识别与触控、且常配合 Cine EI 锁 ISO。
> **解锁**：**先退出 S&Q**(把 S&Q 开关关掉)；若 ISO 仍变灰，说明在 **Cine EI**，切到 **Flexible ISO**；确认**对焦模式为 AF-C** 且"人脸/眼睛优先=开、识别对象=人物"，眼控和触控追踪就回来了。

> **症状**：只有快门不能调，其他正常
> **判断**：多半只是**不在 S/M 档**(在 P/A/Auto)。
> **解锁**：转到 **S 或 M** 档即可。

## 用法提示 / Usage

1. 先按"第 0 步"定位是哪个**模式/格式**在接管，再回答——别上来就让人改一堆设置。
2. 多个参数**同时**变灰，往往是**同一个上游模式**(最常见 S&Q、Cine EI、高帧率、Auto/P 档)造成的，先解决那个根因。
3. 结合机型给**具体菜单名**；拿不准某机型的叫法就说"逻辑是 X，你的机型里大概叫 Y，建议查说明书确认"，不要编造不存在的选项。
