# 后期修图工作流 / Editing Workflow

给修图建议时参考本文件。核心理念三句话：
1. **后期是让照片更接近你当时的感受，不是换一张照片。**
2. **少即是多** —— 新社员最大的坑是过度修图。
3. **前期能解决的(构图、光、对焦)别指望后期补救** —— 后期补救永远有代价(画质损失、显假)。

工具不限(Lightroom / 手机 Snapseed / 醒图 / 泼辣修图都行)，但**调整的先后顺序通用**。给具体照片建议时，把步骤对应到评图时发现的问题上。

## 1. 标准修图顺序 / The standard order

为什么讲顺序？因为后面的调整建立在前面的基础上(比如白平衡没定好，调色就是空中楼阁)。

1. **选片 Cull**
   先扔掉糊的、重复的、闭眼/表情差的，只在有潜力的片子上花时间。修图前先选片，能省一半精力。

2. **裁剪 / 摆正 Crop & Straighten**
   先定构图：摆正地平线/竖线，裁掉边缘干扰物，必要时重新构图(二次构图)。构图决定观众怎么读这张图，所以放最前。

3. **白平衡 White Balance**(颜色的地基)
   先校冷暖：偏黄就降色温(temperature)，偏蓝就升；偏绿/品红调色调(tint)。技巧：用吸管点画面里**本该是中性灰或白**的地方，让软件自动校准，再微调。**所有调色都建立在白平衡正确之上。**

4. **曝光与对比 Exposure & Contrast**
   - 整体亮度(exposure)定基调。
   - **高光 highlights / 白色 whites**：往下压，找回过曝天空、灯光的细节。
   - **阴影 shadows / 黑色 blacks**：暗部死黑就往上提，找回细节(但别提太狠，会发灰发脏)。
   - 对比(contrast)控制整体力量感。
   目标：高光不死白、暗部不死黑、该亮的亮该暗的暗。

5. **色彩 Color grading**
   - **饱和度 saturation / 自然饱和度 vibrance**：优先用 vibrance(它保护肤色、只提不够艳的颜色)，别无脑拉 saturation。
   - **HSL**：单独微调某个颜色的色相/饱和/明度(如把天空调蓝一点、压一下抢眼的杂色)。
   - **色调 / 影调 color grade**：给高光/阴影加冷暖倾向，统一情绪(如阴影加一点青、高光加一点暖)。
   - 服务情绪，不是越浓越好。

6. **局部调整 Local adjustments**
   用蒙版/画笔/渐变/径向只调画面一部分：压暗背景、提亮主体脸、给天空加渐变、加暗角(vignette)把视线收向主体。这一步是让主体"跳出来"的关键。

7. **锐化与降噪 Sharpen & Noise reduction**
   最后做，适量。先降噪(高 ISO 暗光片)再锐化。锐化过头会出"白边"和颗粒；降噪过头会"涂抹"得像塑料。

最后：**和原图对比一眼**。如果改完"一眼假"，往回收。

## 2. 常见问题 → 对应调整 / Problem → fix

把评图发现的问题，翻译成具体动作(给方向和幅度感，不必死磕数值)：

| 评图发现的问题 | 后期对应调整 |
|---|---|
| 整体偏黄 / 偏蓝 | 白平衡：降 / 升色温 |
| 人脸偏黄、肤色不对 | 白平衡微调 + HSL 里橙色通道调整 |
| 天空死白过曝 | 高光、白色往下压；必要时局部渐变压暗天空 |
| 暗部死黑没细节 | 阴影、黑色往上提(适度) |
| 整体灰、不通透 | 适当加对比、压一点黑色、提一点清晰度 |
| 颜色太淡没生气 | vibrance 适度+，而非猛拉 saturation |
| 颜色过浓显假 | 往回收 saturation/vibrance |
| 主体不突出 | 局部压暗周围 + 加暗角 + 提亮主体 |
| 地平线歪 / 楼倾斜 | 裁剪里摆正 + 透视/几何校正 |
| 画面松散、有干扰 | 二次裁剪构图，裁掉边缘杂物 |
| 暗光噪点多 | 降噪适度，纪实片可容忍一些噪点 |

## 3. 过度修图警示信号 / Over-editing red flags

看到这些，提醒往回收：
- 天空蓝得发紫、草绿得发荧光("糖水""塑料感")。
- HDR 感：暗部全提亮导致画面发灰、发脏、不通透、有光晕。
- 磨皮过度，皮肤像蜡/塑料，没有纹理。
- 锐化过头，边缘有白边、噪点被放大。
- 暗角重到像隧道。
- 别人会问"这是不是 P 过头了？"——那就是过了。

---

## 4. 一句话原则 / One-liner

> 修图的最高境界是**别人看不出修过**，只觉得"这张照片真好看、真有感觉"。先把照片修"对"(还原真实、修正缺陷)，再考虑修"美"(风格化调色)。新社员先练好前者。
