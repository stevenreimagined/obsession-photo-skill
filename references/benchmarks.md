# 优秀照片文字标杆 / Reference Standards

点评、选片时用本文件做"标尺"——一张照片在某题材里**接近这些特征**就是好作品。这里用**文字描述**而非内嵌图片(轻量、无版权风险)。配合 `references/masters.md` 的大师风格一起用。

> 用法提示：这些是"理想态"，不是打分清单。新社员的照片不需要全中，越接近越好。点评时可以说"距离优秀作品，这张还差在 ___"。

## 1. 各题材优秀照片特征 / What great looks like

### 活动 / 纪实 Event & Documentary
- 抓住了**关键瞬间**：欢呼的高点、专注的神情、人与人之间的互动、笑到一半的真实表情。
- 画面**交代了环境和氛围**：看得出在什么场合、什么情绪，不只是"一群人站着"。
- 主体清晰、有视觉中心，背景虽乱但不抢戏(或用浅景深虚化)。
- 真实、自然、有故事感，胜过整齐摆拍。
- 容忍轻微噪点/不够锐——瞬间对了就值。
- 对照：Cartier-Bresson 的瞬间秩序、Capa 的"够近"。

### 人像 Portrait
- **眼睛锐利、有眼神光**，视线有内容(看镜头或若有所思都行)。
- **光线柔和**地修饰脸部(窗光、阴天、黄金时刻、反光板),没有难看的硬阴影。
- **肤色自然准确**，不偏黄不偏青。
- 背景**干净或柔和虚化**，不与人物打架。
- 表情/姿态自然放松，传递出人物的性格或情绪。
- 对照：Penn 的极简干净、McCurry 的眼神与自然光。

### 风光 / 自然 Landscape & Nature
- **光线是主角**：黄金时刻的暖光、蓝调时刻的冷调，或戏剧性的光打在主体上的瞬间。
- **构图有层次**：清晰的前景/中景/背景，纵深感强；常有引导线把视线带入画面。
- **明暗层次完整**：高光不死白、暗部有细节(亚当斯式的丰富影调)。
- 地平线水平，全画面清晰锐利(大景深)。
- 色彩自然不过饱，服务于当时的氛围。
- 有时放入一个人/物做比例参照，凸显壮阔。

### 建筑 / 城市 Architecture & City
- **线条横平竖直**，竖线垂直、没有楼体倾斜的透视畸变。
- 利用**对称、重复、几何线条**形成强构图。
- **侧光/蓝调时刻**带来立体感和氛围(华灯初上的城市最佳)。
- 干净、克制，没有杂乱的电线/路人干扰主结构(或刻意纳入做点缀)。
- 常见比例参照(一个人)体现尺度。

### 通用底线 General baseline
一张"合格的好照片"至少满足：**主体清晰可辨 + 曝光正常(不死白不死黑) + 白平衡正确(颜色舒服) + 没有明显干扰物/穿帮 + 有一个能说清楚的看点。**

## 2. 一眼区分 / Good vs ordinary

| 普通照片 | 优秀照片 |
|---|---|
| "记录了一下" | 想表达点什么，有看点 |
| 主体不明、画面乱 | 一眼看到主体，做了减法 |
| 平光、随手拍 | 光线被有意识地利用 |
| 边缘有穿帮/杂物 | 画面干净，边缘讲究 |
| 颜色偏 / 过浓显假 | 白平衡准、色调统一服务情绪 |
| 摆拍僵硬 / 错过瞬间 | 抓到自然真实的瞬间 |
| 后期过度、一眼假 | 后期克制、看不出修过 |

## 3. 社团优秀作品库 / Club benchmark library

> 这一节用来沉淀**本社团自己的优秀作品特征**——这是最贴合你们风格的标尺。
> 当社员提供优秀样片时，在这里追加一条文字描述，记录它好在哪(题材 + 哪几个维度突出 + 一句可学的点)。日积月累，评图标准会越来越"懂"本社的审美。

格式示例：
```
- [2026 迎新晚会 / 纪实] 抓住主持人谢幕鞠躬、台下举手机灯海的瞬间；逆光剪影+暖色灯光，氛围满分。可学：预判高潮时刻、用现场光。
- [社团合影 / 人像] 黄金时刻侧逆光，发丝透光，肤色干净，所有人表情自然。可学：选对时间和机位。
```
