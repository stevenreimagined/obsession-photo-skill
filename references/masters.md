# 摄影大师与可学技术 / Masters & Techniques

讲"什么是好照片"、给作品定标尺时参考本文件。重点不是记名字，而是**从大师身上能学到哪条具体、可模仿的东西**。给新社员讲时，挑 1–2 位相关的就够，别报菜名。

## 目录
1. 纪实 / 街头 / 活动
2. 人像
3. 风光 / 自然
4. 建筑 / 城市
5. 跨题材通用原则

## 1. 纪实 / 街头 / 活动 Documentary · Street · Event

**Henri Cartier-Bresson 亨利·卡蒂埃-布列松**
- 关键词："决定性瞬间 the decisive moment"——画面元素在某一刻恰好形成呼应。
- 可学：用小相机(他用 35mm Leica)轻装、安静地等待，预判动作发生的那一刻按下快门；强调几何构图的瞬间秩序。
- 给新社员：拍活动时**多按几张、提前预判**，比事后后悔"刚才那下没拍到"强。

**Robert Capa 罗伯特·卡帕**
- 名言："如果你拍得不够好，是因为你离得不够近 (If your pictures aren't good enough, you aren't close enough)。"
- 可学：靠近主体、进入现场，画面才有参与感和张力。新社员常因为"不好意思"站太远，导致画面平淡。

**马格南 Magnum 群体的启示**
- 档案里有大量噪点、轻微失焦、过曝的照片仍是经典——**故事和瞬间 > 技术完美**。
- 拍纪实/活动时，先让被摄者放松、习惯你的存在，再等自然发生的瞬间。
- 常用 35mm 或更广的镜头，让观众有"身临其境"感。

## 2. 人像 Portrait

**Irving Penn 欧文·佩恩**
- 可学：极简背景 + 精心布光，把全部注意力交给人物本身。干净的背景是好人像的捷径。

**Steve McCurry 史蒂夫·麦凯瑞**(《阿富汗少女》)
- 可学：眼神的力量 + 柔和自然光 + 饱和但不假的色彩；环境人像里背景交代身份但不喧宾夺主。

**给新社员的人像要点**
- **对焦在眼睛**，眼睛实了照片就"活"。
- 找柔光(窗边、阴影、阴天)，避免正午头顶硬光在脸上压出难看阴影。
- 注意肤色白平衡，别让人脸偏黄或偏青。

## 3. 风光 / 自然 Landscape · Nature

**Ansel Adams 安塞尔·亚当斯**
- 关键词：**区域系统 Zone System**——精确控制从纯黑到纯白的影调，让画面层次极其丰富。
- 可学(简化版)：拍风光要照顾**高光不过曝、暗部有细节**，追求完整的明暗层次；前期慢工出细活(他用大画幅、构图极其讲究)。
- 给新社员：风光不急，**等光**(黄金/蓝调时刻)、用三脚架、求全画面清晰。

**Galen Rowell / 现代风光**
- 可学：把人或动物放进大景里做比例参照，强化"壮阔感"；追逐瞬息的光(光线打在山脊那一刻)。

## 4. 建筑 / 城市 Architecture · City

**通用可学点**
- **横平竖直**：保持竖线垂直、地平线水平，避免楼房"东倒西歪"(透视畸变)，后期可校正。
- **线条与重复**：利用建筑的线条、对称、重复图案做构图骨架。
- **光影**：侧光让结构出立体感；蓝调时刻 + 室内灯光的城市最讨喜。
- **加入比例**：放一个人进画面，体现建筑尺度。

## 5. 跨题材通用原则 / Universal lessons

从所有大师身上反复出现的共性，可以直接转述给新社员：

1. **减法**：好照片通常是去掉多余元素后剩下的。先问"画面里什么可以不要"。
2. **光是主角**：摄影 (photo-graphy) 字面就是"用光作画"。先看光，再看其他。
3. **靠近 / 选对机位**：换个角度、蹲下、靠近，往往比换设备更能救一张照片。
4. **等待与预判**：最好的瞬间需要等，活动里要预判动作。
5. **意图 > 规则**：大师破规则是因为他们知道为什么破。新社员先理解规则背后的原因，再谈打破。
6. **量大出好片**：多拍、多看、多对比，眼睛比手先进步。

> 关于"AI 生成的优秀照片作为基准"：可以让 AI 生成或挑选一些构图/用光教科书式的范例来**演示原则**(如完美的三分法、黄金时刻侧光人像)，作为视觉对照。但要明确告诉新社员——AI 图是"原则示范"，真实摄影的价值在于真实的瞬间、现场和情感，这是 AI 替代不了的。社团自己拍的优秀作品才是最终标杆。
