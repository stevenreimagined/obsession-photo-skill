# 拍摄中 · 现场决策助手 / On-Site Troubleshooting

摄影师在现场直接描述问题，用统一格式快速给排查方案：

> **最可能原因 → 立即操作 → 仍未解决时的下一步**

回答要**短、可立即执行**(现场没时间读长文)。结合用户的机型和当前设置给具体值。下面是常见问题库；遇到没列的，按同样的"原因→操作→下一步"结构推理。

---

## 曝光 / 颜色类

### 画面偏黄 / 偏色
最可能：自动白平衡被舞台暖光/暖色 LED 带偏。
立即操作：白平衡改**手动色温**，先试 **3800–4500K**。
下一步：肤色仍黄 → 用**白平衡偏移(WB Shift)**往蓝/品红方向微调；或拍 RAW 回去统一校。

### 人物脸太暗(逆光/追光)
最可能：相机按整个画面测光，被亮背景拉低了人脸曝光。
立即操作：测光改**点测光**对准人脸，或加 **+1～+2 EV 曝光补偿**。
下一步：仍暗 → 开大光圈/升 ISO；或用反光板/补光灯；高反差场景按人脸曝光、宁让背景过一点。

### 高光死白 / 过曝(白衣、灯、天空)
最可能：整体过曝，高光溢出。
立即操作：减 **-0.7～-1 EV**，看回放**直方图**右侧别贴边；开**斑马线/高光警告**。
下一步：动态范围不够 → 拍 RAW 保高光、暗部后期提；或等光比小一点再拍。

### 画面忽明忽暗(一张张亮度跳)
最可能：用了光圈优先/自动，现场光在变，相机每张重新测光。
立即操作：改 **手动曝光(M) + Auto ISO**，锁死快门光圈。
下一步：仍跳 → 关掉 Auto ISO 改全手动，固定一个 ISO。

## 快门 / 频闪类

### 快门调不动 / 不能再快
最可能：当前是光圈优先或全自动，快门由相机定；或在 Auto ISO 下到了限制。
立即操作：转到 **快门优先(S/Tv)** 或 **手动(M)**，直接设快门值。
下一步：仍受限 → 检查是否电子前帘/闪光同步速度限制；或光线不足导致快门上不去，需开大光圈/升 ISO。

### 室内出现明暗条纹(照片)
最可能：快门速度与灯光闪烁(50/60Hz)不同步。
立即操作：快门改 **1/100 s(50Hz)** 或 **1/120 s(60Hz)**；有机械快门则开**防闪烁拍摄**。
下一步：纯电子快门机型(如 FX30)无防闪烁 → 只能锁在 1/50、1/100 这类安全值。

### LED 背景屏出现滚动横条(视频)
最可能：屏幕刷新率与你的快门/帧率不匹配。
立即操作：快门设 **1/50 或 1/100**；帧率按 50Hz 用 **25/50p**。
下一步：仍有条 → 微调快门角度；或改变机位角度/距离；实在不行后期降条纹。

## 对焦 / 清晰度类

### 演员移动时总是失焦
最可能：用了单次对焦(AF-S)，或对焦区域太大追到了背景。
立即操作：改 **连续对焦 AF-C** + **人物/眼部识别**；对焦区域选**中等/区域**而非全画面。
下一步：仍跑焦 → 提高快门、收一点光圈加景深；暗光下开对焦辅助、找高对比边缘。

### 照片整体发糊(不是失焦)
最可能：快门太慢导致手抖/主体动糊。
立即操作：快门提到 **≥1/(等效焦距)**，动作场景 ≥1/500；开机身/镜头防抖。
下一步：仍糊 → 上三脚架/独脚架；检查是否对焦点没落在主体。

### 暗光下对不上焦 / 拉风箱
最可能：光线和对比不足，相机找不到合焦点。
立即操作：把对焦点放到**主体边缘高对比处**(如肩线、眼镜框)；开对焦辅助灯。
下一步：仍不行 → 手动对焦 + 峰值对焦(focus peaking)放大确认。

## 用法提示 / Usage

回答现场问题时：
1. 用"最可能原因 → 立即操作 → 下一步"三段式，别长篇大论。
2. 结合用户报的**机型和当前设置**给**具体数值**(如"你 FX30 没机械快门，改 1/100")。
3. 一次先给最可能的一条路径，让他试；不奏效再给下一步。现场决策讲究快。
