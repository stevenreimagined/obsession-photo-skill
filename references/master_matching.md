# 对标大师 / Match to a Master

点评一张照片时，除了指出好坏，再多做一步：**判断这张照片的"路子"最接近哪位大师/哪种风格，并告诉社员该去研究谁**。这能给新社员一个具体的学习方向("你这张有 XX 的味道，去看他的画册"），比空泛表扬有用得多。

配合 `masters.md`(大师与技术) 一起用。本文件提供**匹配方法**。

## 怎么匹配 / How to match

先看照片**最突出的一个特征**(不是面面俱到)，再对到下表。一张照片只匹配 1 位、最多 2 位，别硬凑。匹配后用一句话说清：**像谁 → 因为什么 → 去研究他的什么**。

| 照片的突出特征 | 最接近的大师 / 风格 | 让社员去研究什么 |
|---|---|---|
| 街头/活动里抓到几何与瞬间恰好重合 | Henri Cartier-Bresson 布列松 | "决定性瞬间"、等待与预判、几何构图 |
| 冲进现场、贴得很近、有张力的纪实 | Robert Capa 卡帕 / 马格南 | "离得不够近"、参与感、35mm 广角 |
| 戏剧性光线(火光/逆光/暗调)的纪实人文 | 马格南纪实 / Steve McCurry | 用现场光讲故事、暗调氛围、暖色调 |
| 眼神有力、柔光、色彩饱满的环境人像 | Steve McCurry 麦凯瑞 | 眼神光、自然柔光、环境交代身份 |
| 极简背景、布光讲究、只剩人物本身 | Irving Penn 佩恩 / Avedon | 干净背景、棚拍布光、做减法 |
| 黑白、影调从纯黑到纯白极丰富的风光 | Ansel Adams 亚当斯 | 区域系统、影调层次、慢工构图 |
| 等到极致光线、壮阔自然 + 比例参照 | Galen Rowell / 现代风光 | 追光、黄金/蓝调时刻、用人做比例 |
| 横平竖直、对称、重复线条的建筑 | 建筑/极简几何流派 | 透视校正、线条与图案、侧光立体 |
| 高饱和撞色、平面化构图、设计感 | 现代色彩/商业风(如 Saul Leiter 街头彩色) | 撞色、负空间、把画面当平面设计 |
| 安静、留白、含蓄情绪的日系/小清新 | 滨田英明 / 日系风格 | 柔和色调、留白、生活化瞬间 |

> 表里没有完美对应时，就近取一个并说明差异("有 X 的味道，但光再硬一点会更像")。重点是给方向，不是贴标签。

## 输出示例 / Example phrasing

> 🎭 **对标大师**：这张讲者 + 火光投影的暗调纪实，路子接近**马格南的纪实人文**和 **Steve McCurry** ——都靠现场的戏剧性光线讲故事、暖色调出氛围。建议去研究 McCurry 怎么用一束光打亮主体的脸、怎么让暖色不显脏；以及马格南摄影师"如何在不打扰现场的情况下等到关键瞬间"。

## 给新社员的提醒 / Note

对标大师是**学习地图**，不是要你模仿成谁。看大师是为了偷到具体的"招"(怎么用光、怎么等瞬间、怎么构图)，再用到自己的题材上。最终目标是形成**本社团自己的风格**(见 `benchmarks.md` 的作品库)。
