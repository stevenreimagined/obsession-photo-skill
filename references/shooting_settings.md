# 拍前参数推荐 / Pre-Shoot Settings Advisor

根据**场景 + 光线 + 机型**给出整套相机参数，并**解释为什么**。核心原则：不只给数字，要让摄影师理解取舍，现场能自己微调。

可配合 `scripts/recommend_settings.py` 输出参数卡。本文件是方法论 + 预设库 + 机型/防频闪知识。

## 1. 核心方法论 / How to reason

**先定优先级，再配三角。** 曝光三角(快门/光圈/ISO)是此消彼长，先问"这个场景最不能丢的是什么"：

- **抓动作**(运动、舞台、活动抓拍)→ 快门优先。先锁住够快的快门，光圈尽量开大，ISO 交给 Auto。**理由：运动模糊基本没法后期救，而噪点后期可降。**
- **要虚化/氛围**(人像、采访)→ 光圈优先(大光圈)，快门够用即可。
- **要全清晰**(合影、风光、建筑)→ 光圈收到 f/5.6–f/8，用三脚架补快门。

**手持最低快门**：约 1/(等效焦距) 秒，有防抖可再放宽 2–3 档。拍人物动作另算(见下)。

**活动摄影黄金组合**：**手动曝光(M) + Auto ISO**。你手动锁定快门和光圈(保证动作和景深稳定)，把亮度波动交给 Auto ISO，并设好 ISO 上限。这样在明暗变化的现场，画面不会忽明忽暗，又不丢动作。可配合曝光补偿微调。

**测光与保高光**：宁可稍欠一点保住高光(白衣、灯、天空),暗部后期可提；高光过曝死白救不回。RAW 拍摄能给暗部更多提亮空间。

**对焦**：动态主体用连续对焦(AF-C/AI Servo) + 主体/眼部识别；静态用单次(AF-S)。会摆动失焦多半是对焦模式不对或对焦区域太大。

## 2. 防频闪 / Anti-flicker（室内 & LED 屏必读）

人造光(LED、荧光灯、舞台灯)按市电频率闪烁。快门太快或与频率不同步，照片会出现**明暗条纹/整张忽明忽暗**，视频里 LED 大屏会出现**滚动横条**。

**市电频率**：中国/欧洲/大部分亚洲 = **50Hz**；北美/日本东部 = **60Hz**。

**拍照(防条纹的安全快门)**
- 50Hz 地区：用 **1/50 或 1/100 秒**(及其整数关系)最稳；尽量避开 1/125、1/250 这类与 100Hz 不同步的值，劣质 LED 下易出条纹。
- 60Hz 地区：用 **1/60 或 1/120 秒**。
- 相机若有**防闪烁拍摄(Anti-flicker Shooting)**功能(需机械快门)，开启它，相机会在闪烁波峰释放快门。**纯电子快门机型(如 Sony FX30)没有此功能，只能靠选对快门速度。**

**拍视频**
- 帧率跟市电：50Hz 用 **25p/50p**，60Hz 用 **30p/60p**，能从源头减少闪烁(24p 在 50Hz 灯下偶有轻微闪)。
- 快门角度 **180°**(快门速度 = 1/(2×帧率))；遇 LED 屏滚动条，微调快门(如 1/50、1/100)或降帧率同步。
- 实在拍不掉的 LED 大屏条纹：靠近/正对屏幕、降快门、或后期处理；条纹多源于屏幕刷新率与快门不匹配。

## 3. 场景预设库 / Scene presets

数值是**起点**，按现场光线和机型微调。WB 一律建议**手动设色温**(避免自动白平衡在混合光下忽冷忽暖)。

### 室内运动 / 篮球(光线一般)
- 快门：**1/500 s 起**(凝固跳跃/快攻，光暗可降到 1/320 保底)
- 光圈：**尽量最大**(f/2.8 或镜头极限)
- ISO：**Auto，上限 12800**
- 白平衡：**手动**(常见体育馆 4000–5000K)，避免连续变色
- 对焦：**AF-C + 人物/眼部识别**
- 连拍：**Hi+**
- 格式：**RAW**(暗光宽容度) 或 RAW+JPEG
- 防频闪：体育馆 LED 多，开防闪烁；纯电子快门则用 1/500 时留意条纹，必要时回到 1/100×倍数

### 室内活动 / 讲座 / 会议
- 快门：**1/160–1/250 s**(人讲话手势不糊)
- 光圈：f/2.8–f/4
- ISO：Auto，上限 6400
- 白平衡：手动，按现场灯(暖光 3800–4500K)
- 对焦：AF-C + 眼部识别
- 连拍：中速
- 格式：RAW+JPEG
- 防频闪：开防闪烁 / 50Hz 用 1/100×倍数

### 舞台演出 / 话剧 / 晚会(追光、高反差)
- 快门：**1/250 s 起**(演员走动)，独舞静态可 1/160
- 光圈：尽量大(f/2.8)
- ISO：Auto，上限 12800；**按追光下的人脸测光，宁欠勿过**(保住高光)
- 白平衡：手动(舞台暖光 3200–3800K，别用 AWB)
- 对焦：AF-C + 眼部识别，对焦区域选中等(避免追到背景)
- 连拍：Hi
- 格式：RAW(高反差需要宽容度)
- 防频闪：舞台 LED/频闪灯多，开防闪烁；电子快门用 1/100×倍数

### 室外活动 / 运动会(白天)
- 快门：动作 **1/1000 s**，人物常规 1/500
- 光圈：f/4–f/5.6(留点景深给抓拍)
- ISO：Auto，上限 1600(白天够用)
- 白平衡：晴天 5200–5600K / 阴天 6500K(或手动)
- 对焦：AF-C + 人物识别
- 连拍：Hi+
- 格式：RAW+JPEG
- 防频闪：户外自然光通常无需

### 合影 / 团体
- 快门：1/200 s 以上(人多防个别人晃)
- 光圈：**f/5.6–f/8**(保证前后排都实)
- ISO：尽量低(100–400)
- 白平衡：按光线手动
- 对焦：AF-S，对焦点放在**第一排人脸**
- 连拍：低速连拍(防闭眼，多拍几张)
- 格式：RAW+JPEG

### 人像 / 采访
- 快门：1/200 s(配合焦距，防抖)
- 光圈：**大光圈 f/1.4–f/2.8**(虚化、突出人物)
- ISO：尽量低
- 白平衡：手动，**保肤色准确**
- 对焦：AF-C + **眼部识别**(眼睛必须实)
- 连拍：单张/低速
- 格式：RAW

### 暗光 / 夜景活动
- 快门：能多慢就多慢但**不低于 1/125**(防主体动糊)；静态场景上三脚架可更慢
- 光圈：最大
- ISO：Auto，上限 12800–25600(接受噪点，运动模糊更难救)
- 白平衡：手动
- 对焦：AF-C + 眼部识别；暗光对焦难，可开对焦辅助灯/选高对比边缘对焦
- 格式：RAW
- 防频闪：城市灯光多为 LED，用 1/100×倍数

## 4. 机型补充说明 / Camera notes

不同机型菜单叫法不同，但逻辑通用。几点常见差异：

- **Sony FX30**：APS-C 电影机，**双原生 ISO(基准约 800 / 2500，S-Log3 下)**——暗光优先把 ISO 设在第二基准(约 2500)附近能更干净。**只有电子快门、无机械快门**，所以**没有"防闪烁拍摄"**，室内务必靠快门速度(50Hz→1/50、1/100)避条纹。视频常用 S-Log3(Cine EI)留后期空间，快出片可用 **S-Cinetone**(直出讨喜)。视频帧率按 50Hz 选 25/50p。
- **Sony α 系列(α7/α9 等)**：有机械快门+防闪烁拍摄，室内开它；Auto ISO 可设最低快门(Auto ISO Min SS)。
- **Canon(R 系列)**：防闪烁拍摄 = "防闪烁拍摄"；连续对焦 = AI Servo / 伺服;主体识别在 AF 菜单。
- **Nikon(Z 系列)**：连续对焦 = AF-C；防闪烁 = "闪烁消减"。
- **Picture Profile / 直出风格**：要后期调色 → Log(S-Log3/C-Log/N-Log)或 HLG；要快出片直接用 → 标准/中性/电影风格(如 S-Cinetone)。**新社员活动快出片，建议直出风格 + RAW 双保险，别一上来全 Log。**

> 拿不准某机型的具体菜单项时，明确告诉用户"逻辑是 X，你的机型里大概叫 Y，建议查一下说明书确认"，不要编造不存在的功能。

## 5. 输出格式 / Output format

给参数推荐时，**先一句话定策略**，再列参数(每项附一句为什么)，最后给 1–2 条现场提醒。例如：

```
🎬 场景：室内篮球比赛 · FX30 · 灯光一般
策略：快门优先保证动作清晰——运动模糊救不回，噪点能后期降。

· 快门：1/500 s 以上 —— 凝固跳跃和快攻的瞬间
· 光圈：尽可能开到最大(f/2.8) —— 多进光、压低 ISO
· ISO：Auto ISO，上限 12800 —— 把亮度波动交给相机，保动作
· 白平衡：手动设色温(约 4500K) —— 避免画面连续变色
· 对焦：AF-C + 人物/眼部识别 —— 跟住移动球员
· 连拍：Hi+ —— 一串里挑动作最完整的一张
· 格式：RAW —— 暗光下暗部宽容度更好
· 防频闪：FX30 无机械快门，注意 LED 条纹，必要时回到 1/100 的整数倍

⚠️ 现场：先对一次白平衡基准；若 ISO 频繁顶到上限，说明光线太暗，优先降快门到 1/320 而不是无限拉 ISO。
```
