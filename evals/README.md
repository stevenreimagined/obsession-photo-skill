# 触发测试 / Trigger Eval

`trigger_eval.json` 是一组用来检验 skill **触发准确度**的查询：10 条**应触发**(真实社团场景的不同说法) + 10 条**不该触发**的近似干扰项(和摄影沾边、但本 skill 不该接的活)。

## 怎么跑自动评测(在你本地、已登录的 Claude Code 里)

沙盒环境里 `claude` CLI 未登录，无法自动评测；请在你自己电脑、登录后的 Claude Code 中用 skill-creator 跑：

```bash
# 单次评测当前描述的触发率
python3 -m scripts.run_eval \
  --eval-set <本仓库>/evals/trigger_eval.json \
  --skill-path <本仓库> --runs-per-query 3

# 或自动优化描述(5轮，挑测试集表现最好的)
python3 -m scripts.run_loop \
  --eval-set <本仓库>/evals/trigger_eval.json \
  --skill-path <本仓库> --model <你的模型> --max-iterations 5 --verbose
```
